# 大师拷贝 v2.1.0 发布说明

本次版本聚焦稳定性与使用体验提升，建议所有用户更新。

## 新增
- 统一的构建/发布 CLI（内部改进，不影响日常使用）

## 改进
- 上传进度事件新增 percentage 字段，进度条显示更流畅
- Electron 启动顺序优化，提升应用稳定性

## 修复
- 若干边界情况下的扫描/上传问题

## 已知问题
- 首次运行（或使用自签名包）可能被 Gatekeeper 拦截：请在 Finder 中右键应用并选择“打开”，或在系统设置 → 安全性与隐私中允许运行

## 安装方法
1. 下载 DMG 安装包
2. 打开 DMG 文件
3. 将 大师拷贝 拖到“应用程序”
4. 首次运行若提示安全警告，右键应用选择“打开”

## 系统要求
- macOS 11.0 或更高版本
- 约 100MB 可用磁盘空间
